Imágenes para una Web Accesible

***A tener en cuenta para incluir textos lternativos adecuados y accesibles a las imágenes de tu web

  • ➥ La imagen no debe ser ni forma parte de un enlace o botón.
  • ➥ La imagen no es decorativa, sino que debe aportar información adicional.
  • ➥ La imagen no debe ser un cuadro ni otra expresión artística cuya función principal sea crear una experiencia sensorial que las palabras no pueden explicar.
  • ➥ Que la imagen no forme parte de una serie de imágenes contiguas que transmitan una información de forma conjunta.
  • ➥ Que la información que proporciona la imagen se puede describir en menos de 100-150 caracteres. Sea claro, conciso

Según el W3C,indica que la longitud máxima del atributo ALT será de 150 caracteres. Sin embargo otras fuentes, indican no sobrepasar los 100 caracteres. Si el texto alternativo que describe la información que transmite la imagen sobrepasa los 100 caracteres, pregúntate, ¿puedo reducirlo? ¿sería mejor una descripción extensa?

****Ejemplos

Caso 1: La imagen solo contiene texto

alt=”texto de la imagen”


Debes incluir el atributo ALT a la imagen para ser así el texto de la imagen.

El código HTML debería ser: <"img src="images/5anosgarantia.jpg" alt="5 años de garantía"/>

Caso 2: La imagen incluye texto

alt=”información que transmite la imagen, debe incluir el texto”


El texto alternativo para el Atributo "ALT" debe transmitir la misma información que la imagén e incluir el texto presente en la imagen

El código HTML debería ser: <"img src="images/telAtencionCliente_es.gif" alt="Servicio de Atención al Cliente. Teléfono 902 180 074"/>

Caso 3: La imagen no tiene texto

alt=”información que transmite la imagen”


El texto alternativo debe transmitir la mismo información que dicha imagen.

El código HTML debería ser:<"img src="images/noticia.jpg" alt="El secretario de estado estadounidense John Kerry y el viceprimer ministro chino Wang Yang se dan un apretón de manos."/>

Caso 3: La imagen no tiene texto

alt=”información que transmite la imagen”


En este caso, el icono que precede al texto no es decorativo, sino informativo, añade información adicional al texto que le sigue: es una advertencia importante.

El código HTML debería ser:<"p><"img src="images/advertencia.jpg" alt="¡Advertencia!"/> Tu sesión está a punto de expirar.<"/p>

***Recomendaciones generales

  • ➥ El atributo ALT y el atributo TITLE tienen funciones diferentes.
    El atributo ALT ofrece el texto alternativo de la imagen, para transmitir la info, y será este el que lea el lector de pantalla o el que aparezca cuando la imagen no se cargue.
    El atributo TITLE es opcional, sirve para incluir información complementaria (la fecha, el autor, etc) pero nunca debe incluir la misma información que el atributo ALT ni incluir información obvia o importante.
  • ➥El atributo ALT y el pie de foto también tienen una función diferente. No repitas la misma información en ambos.
  • ➥El objetivo no es describir los detalles gráficos de la imagen, sino que sirva para transmitir la misma información que la imagen. Pregúntate, ¿por qué está aquí la imagen? ¿qué información transmite? ¿cuál es su propósito? si no puediera usar la imagen ¿qué texto usaría para transmitir la misma información y/o función?
